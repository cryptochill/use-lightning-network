Use Lightning Network

Scalable, Instant Bitcoin/Blockchain Transactions

What is Lightning Network?

Bitcoin’s Lightning Network is a technical innovation that allows its users to send bitcoin faster and more cheaply than if they were using Bitcoin’s primary network.

How to use Lightning Network?

To start making and accepting payments on Bitcoin Lightning Network you need to install a wallet with Bitcoin Lightning Network support.

Custodial Wallets

When using custodial wallets, the wallet company has the ownership of your keys, which means you have to trust them not to run away with your money.

Custodial wallets offer quick and simple on-boarding, which is great for beginners and as first introduction into Lightning Network.
